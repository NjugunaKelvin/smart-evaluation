## SmartEval Platform 
[Visit the Web](https://smart-evaluation.netlify.app/)
SmartEval is a dedicated opportunities portal to allows businesses and companies access opportunities
like tenders, contracts, partnerships etc.
The system is created in a way to accomodate 500+ sectors in business.

## Features and Capabilities
- Posting of opportunities
- Access to opportunities
- Algrorithm based opportunities feed.
- Submission analytics for opportunities providers.
- Dedicated dashboard for application tracking

## Technologies
Next.js for frontend
